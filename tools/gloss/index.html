User Guide

This tool converts glossed examples into LaTeX code for use in academic papers. The tool supports two glossing styles: GB4E and Linguex. Select the desired style from the dropdown menu before generating the LaTeX code.

To use the tool, simply enter your glossed examples in the textarea provided, select the desired glossing style, and click the "Generate" button. The resulting LaTeX code will be displayed in the output area below.

This tool automatically parses and converts most text to gloss style without any need for special formatting.

You can copy the generated LaTeX code to your clipboard by clicking the "Copy to Clipboard" button.

Special Characters and Gloss-Input

To write the special belted-L character "ɬ" without copy-and-paste, simply type an exclamation mark "!". All non-final exclamation marks will be automatically converted.

In your gloss line, all items to be glossed should be in ALL CAPS.

Sub-Examples

This tool automatically parses and converts text labeled with subexample markers "a.", "b.", "c.", etc. into LaTeX subexamples. Colons ":" and periods "." after the subexample lables are recognized.

Compare the example input-output below.

a. kati-k, palki-:y-o-k ifa yaam-osi
cat-NOM fast-Y-O-SS1 dog be.like-DIM
``The cat is as fast as the dog.''
b: iisa tabooli-:y-o-k itto chaaha yaam-osi-n tabooli-hchi.
house build-Y-O-NOM tree tall be.like-DIM-DS build-IPFV\\
``He who built a house build it as tall as a tree''

\ex. \a. \gll kati-k, palki-:y-o-k ifa yaam-osi \\
cat-\textsc{nom} fast-\textsc{y}-\textsc{o}-\textsc{ss}1 dog be.like-\textsc{dim}\\
\glt ``The cat is as fast as the dog.''
\b. \gll iisa tabooli-:y-o-k itto chaaha yaam-osi-n tabooli-hchi. \\
    house build-\textsc{y}-\textsc{o}-\textsc{nom} tree tall be.like-\textsc{dim}-\textsc{ds} build-\textsc{ipfv}\\
\glt ``He who built a house build it as tall as a tree''

Multiple Examples

The tool can output multiple examples at a time! Simply separate examples by at least one empty line.

``The cat is as fast as the dog.''

Question and Answer Pairs

Examples prefaced with a Q/A label Q:, A:, S:, etc. will be automatically parsed for any label.

Q: Chi-holchifa-k naas-oolĩ?
2SG-name-NOM what-EX.M.Q
`What's your name?'
A: Cha-holchifa-ya Mary
1SG-name-TOP Mary
`My name is Mary.'

\ex. \a. Q: \gll  Chi-holchifa-k naas-oolĩ? \\
\textsc{2sg}-name-\textsc{nom} what-\textsc{ex}.\textsc{m}.\textsc{q}\\
\glt `What's your name?'
\b. A: \gll  Cha-holchifa-ya Mary\\
\textsc{1sg}-name-\textsc{top} Mary\\
\glt `My name is Mary.'

Adding Contexts and other top-line content

Contexts and other top-line content can be added by wrapping the text in [square brackets]. These will be deleted in the output text, so text that should be in square brackets should have two pairs of square brackets.

[[CONTEXT: Asking a question.]]
Chi-holchifa-k naas-oolĩ?
2SG-name-NOM what-EX.M.Q
`What's your name?'

\ex. [CONTEXT: Asking a question.]
\gll Chi-holchifa-k naas-oolĩ? \\
\textsc{2sg}-name-\textsc{nom} what-\textsc{ex}.\textsc{m}.\textsc{q}\\
\glt `What's your name?'
